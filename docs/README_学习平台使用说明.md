# 系统架构设计师学习平台使用说明

## 📚 项目简介

这是一个专为系统架构设计师考试打造的在线学习平台，提供完整的章节内容、重点提纲和必背内容。

## 🚀 快速开始

### 方法1：直接打开（推荐）

1. 双击打开 `index.html` 文件
2. 选择"章节学习"开始学习
3. 在左侧导航栏选择要学习的章节

### 方法2：使用本地服务器

如果遇到跨域问题，可以使用本地服务器：

```bash
# 使用Python 3
python -m http.server 8000

# 使用Python 2
python -m SimpleHTTPServer 8000

# 使用Node.js (需要先安装http-server)
npx http-server -p 8000
```

然后在浏览器访问：`http://localhost:8000`

## 📖 功能说明

### 1. 主页 (index.html)

- 考试类型选择
- 快速导航到不同学习模式
- 平台功能介绍

### 2. 章节学习页面 (chapters.html)

#### 三种学习模式：

**📖 完整章节**
- 查看完整的教材内容
- 适合系统学习和深入理解

**⭐ 重点提纲**
- 精心整理的重难点知识
- 包含详细的公式、算法和例题
- 适合复习和强化记忆

**🎯 必背内容**
- 核心公式、概念、算法
- 快速查询和考前突击
- 适合考前冲刺

#### 功能特性：

- **章节导航**：左侧侧边栏显示所有章节
- **搜索功能**：快速查找章节
- **星级标注**：显示章节重要程度
- **打印功能**：支持打印当前章节
- **下载功能**：下载Markdown源文件
- **响应式设计**：支持手机、平板、电脑

## 💡 使用技巧

### 1. 快速定位

- 使用搜索框快速查找章节
- 收藏常用章节（浏览器书签）
- 使用URL参数直接访问：
  ```
  chapters.html?type=architect&view=keypoint&chapter=03
  ```

### 2. 高效学习

- 先看重点提纲，了解框架
- 再看完整章节，深入理解
- 最后看必背内容，强化记忆

### 3. 移动学习

- 手机访问时点击左上角"☰"打开目录
- 支持离线使用（下载整个文件夹）

## 🔧 自定义配置

### 修改章节信息

编辑 `config.js` 文件：

```javascript
const CONFIG = {
    examTypes: {
        architect: {
            chapters: [
                { 
                    id: '01', 
                    name: '计算机硬件', 
                    stars: 4,
                    hasKeypoint: true,
                    hasMustKnow: false
                },
                // 添加更多章节...
            ]
        }
    }
};
```

### 添加新的考试类型

在 `config.js` 中添加新的考试类型配置，然后在 `index.html` 中添加对应的入口。

## 🐛 常见问题

### Q1: 页面显示空白或加载失败？

**A:** 可能是跨域问题，请使用本地服务器运行（见"快速开始"部分）。

### Q2: 图片无法显示？

**A:** 确保 `images` 文件夹在同一目录下，且图片路径正确。

### Q3: Markdown格式显示异常？

**A:** 确保网络连接正常，页面需要加载CDN资源（marked.js和github-markdown.css）。

### Q4: 某些章节提示"暂无内容"？

**A:** 该章节的对应视图内容尚未整理，请切换到"完整章节"模式查看。
