## 🚀 新的使用方式

### 方法1：使用启动脚本（推荐）

**Windows:**
```bash
双击 start_server.bat
```

**Linux/Mac:**
```bash
./start_server.sh
```

脚本会自动：
1. 检测Python环境
2. 启动HTTP服务器（端口8000）
3. 打开浏览器访问 `http://localhost:8000/web/`
4. 如果没有Python，尝试直接打开 `web/index.html`

### 方法2：手动启动服务器

```bash
python -m http.server 8000
# 然后访问 http://localhost:8000/web/
```

### 方法3：直接打开

```bash
双击 web/index.html
```

**注意：** 某些浏览器可能有跨域限制，建议使用方法1或方法2
