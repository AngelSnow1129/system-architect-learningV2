# 📱 系统架构设计师学习平台 - 使用演示

## 🎬 功能演示

### 1. 启动平台

#### Windows用户
```
1. 找到 start_server.bat 文件
2. 双击运行
3. 等待浏览器自动打开
4. 看到主页即启动成功
```

#### Mac/Linux用户
```bash
1. 打开终端
2. cd 到项目目录
3. 运行: chmod +x start_server.sh
4. 运行: ./start_server.sh
5. 浏览器自动打开主页
```

#### 直接打开（无需服务器）
```
1. 找到 index.html 文件
2. 双击打开
3. 开始使用
```

### 4. 三种学习模式对比

#### 📖 完整章节模式
```
内容：完整的教材内容
适用：系统学习、深入理解
特点：内容详细、例题丰富

使用场景：
- 第一次学习某个章节
- 需要深入理解概念
- 查看详细的例题解析
```

#### ⭐ 重点提纲模式
```
内容：精心整理的重难点
适用：复习、强化记忆
特点：结构清晰、重点突出

使用场景：
- 复习已学过的内容
- 考前强化重点
- 快速回顾知识框架
```

#### 🎯 必背内容模式
```
内容：核心公式、概念、算法
适用：考前冲刺、快速查询
特点：精简高效、易于记忆

使用场景：
- 考前最后冲刺
- 快速查询公式
- 强化核心概念
```

### 10. 实用技巧

#### 技巧1：快速跳转
```
使用URL参数直接访问特定章节：

访问第3章的重点提纲：
chapters.html?type=architect&view=keypoint&chapter=03

访问第5章的必背内容：
chapters.html?type=architect&view=mustknow&chapter=05
```

#### 技巧2：书签收藏
```
将常用页面添加到浏览器书签：
- 主页
- 常用章节
- 必背索引
- 公式速查
```

#### 技巧3：打印学习
```
1. 打开需要打印的章节
2. 点击"打印"按钮
3. 选择"保存为PDF"
4. 可以在平板或手机上阅读
```

#### 技巧4：离线使用
```
1. 下载整个项目文件夹
2. 复制到U盘或移动硬盘
3. 在任何电脑上都可以使用
4. 无需网络连接（除CDN资源）
```

#### 技巧5：多设备同步
```
方法1：使用云盘同步
- 将项目放在OneDrive/Dropbox
- 自动同步到所有设备

方法2：使用Git
- 上传到GitHub私有仓库
- 在不同设备上clone
```

### 11. 学习建议

#### 第一周：熟悉平台
```
Day 1-2: 浏览所有章节，了解考试范围
Day 3-4: 使用完整章节模式学习1-2章
Day 5-6: 尝试重点提纲模式复习
Day 7: 查看必背内容，了解核心考点
```

#### 第2-8周：系统学习
```
每周学习2-3个章节：
- 周一至周三：完整章节学习
- 周四至周五：重点提纲复习
- 周六：做练习题
- 周日：总结和回顾
```

#### 考前2周：强化复习
```
Week 1: 使用重点提纲过一遍所有章节
Week 2: 使用必背内容强化记忆
```

#### 考前3天：冲刺
```
Day 1: 必背内容 + 公式速查
Day 2: 错题回顾 + 薄弱章节
Day 3: 放松心态 + 快速浏览
```
