# 🚀 快速开始 - 系统架构设计师学习平台

## 📖 三步开始学习

### 第一步：启动平台

选择以下任一方式：

#### 🎯 方式1：一键启动（最简单）

**Windows用户：**
```
双击 start_server.bat
```

**Mac/Linux用户：**
```bash
chmod +x start_server.sh
./start_server.sh
```

✅ 脚本会自动打开浏览器，访问 `http://localhost:8000/web/`

#### 🎯 方式2：直接打开

```
双击 web/index.html
```

⚠️ 注意：某些浏览器可能有跨域限制

#### 🎯 方式3：手动启动服务器

```bash
python -m http.server 8000
```

然后在浏览器访问：`http://localhost:8000/web/`

### 第二步：选择学习模式

进入主页后，选择适合您的学习模式：

#### 📖 完整章节

## ❓ 常见问题

### Q1: 页面显示空白？
**A:** 使用启动脚本运行，或手动启动HTTP服务器

### Q2: 图片无法显示？
**A:** 确保 `images` 文件夹在项目目录中

### Q3: 找不到某个章节？
**A:** 使用搜索功能，或查看快速导航

### Q4: 如何打印学习？
**A:** 点击章节页面右上角的"打印"按钮

### Q5: 可以离线使用吗？
**A:** 可以，但需要先加载CDN资源（marked.js和CSS）

## 💪 学习建议

### 时间分配
- **综合知识**：40%（选择题）
- **案例分析**：30%（问答题）

### 学习方法
1. **理解为主**：不要死记硬背
2. **多做练习**：理论结合实践
3. **定期复习**：巩固记忆
4. **总结归纳**：建立知识体系

### 考试技巧
1. **选择题**：排除法、关键词法
2. **案例分析**：分点作答、条理清晰
3. **论文**：提前准备、多练习
