## 🚀 使用方法

### 方法1：使用启动脚本（推荐）
**Windows:**
```bash
双击 start_server.bat
```

**Linux/Mac:**
```bash
chmod +x start_server.sh
./start_server.sh
```

脚本会自动启动服务器并打开浏览器访问 `http://localhost:8000/web/`

### 方法2：手动启动服务器
```bash
python -m http.server 8000
# 然后访问 http://localhost:8000/web/
```

### 方法3：直接打开（最简单）
1. 双击 `web/index.html`
2. 开始学习

**注意：** 某些浏览器可能有跨域限制，建议使用方法1或方法2

### 方法3：在线部署
可以部署到：
- GitHub Pages
- Netlify
- Vercel
- 任何静态网站托管服务

## 🐛 已知问题和解决方案

### 问题1：跨域限制
**现象**: 直接打开HTML文件时，某些浏览器可能无法加载Markdown文件

**解决方案**:
- 使用本地HTTP服务器
- 运行 `start_server.bat` 或 `start_server.sh`

### 问题2：图片不显示
**现象**: Markdown中的图片无法显示

**解决方案**:
- 确保 `images` 文件夹在同一目录
- 检查图片路径是否正确
- 使用相对路径

### 问题3：CDN资源加载慢
**现象**: 首次加载时Markdown样式延迟

**解决方案**:
- 等待CDN资源加载完成
- 或下载CDN资源到本地
