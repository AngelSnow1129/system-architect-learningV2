## 🚀 使用方法

### 最简单的方式

**Windows:**
```bash
双击 start_server.bat
```

**Linux/Mac:**
```bash
./start_server.sh
```

**直接打开:**
```bash
双击 web/index.html
```

### 访问地址
- **主页**: http://localhost:8000/web/
- **快速导航**: http://localhost:8000/web/nav.html
- **系统诊断**: http://localhost:8000/web/diagnostic.html

## 🎯 学习路径

### 第一阶段：系统学习（1-2个月）
1. 使用"完整章节"模式
2. 按顺序学习每个章节
3. 重点关注5星章节：
   - 02_操作系统知识
   - 03_数据库系统
   - 05_计算机网络
   - 10_软件工程
   - 11_面向对象技术
   - 13_系统架构设计

### 第二阶段：强化复习（2-3周）
1. 使用"重点提纲"模式
2. 重点复习核心知识点
3. 做大量练习题
4. 整理错题和难点

### 第三阶段：考前冲刺（1周）
1. 使用"必背内容"模式
2. 快速过一遍所有公式和概念
3. 查漏补缺
4. 模拟考试
