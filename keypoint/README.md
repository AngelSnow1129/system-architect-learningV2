## 💡 学习建议

### 学习方法
1. **理解为主**：理解原理，不要死记硬背
2. **多做练习**：特别是计算题
3. **总结归纳**：制作思维导图
4. **定期复习**：遵循遗忘曲线
5. **真题演练**：做近5年真题

### 记忆技巧
1. **使用口诀**：每章都有记忆口诀
2. **对比记忆**：相似概念对比
3. **联想记忆**：建立知识联系
4. **图表记忆**：用表格辅助

### 考试技巧
1. **时间分配**：选择题90分钟，案例题90分钟
2. **先易后难**：先做会做的题
3. **检查验算**：计算题要验算
4. **关键词法**：看关键词判断
5. **排除法**：不确定用排除法
