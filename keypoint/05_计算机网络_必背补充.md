### TCP vs UDP（必背）⭐⭐⭐⭐⭐
```
┌──────────┬──────────┬──────────┐
│特征      │ TCP      │ UDP      │
├──────────┼──────────┼──────────┤
│连接      │ 面向连接 │ 无连接   │
├──────────┼──────────┼──────────┤
│可靠性    │ 可靠     │ 不可靠   │
├──────────┼──────────┼──────────┤
│速度      │ 慢       │ 快       │
├──────────┼──────────┼──────────┤
│开销      │ 大       │ 小       │
├──────────┼──────────┼──────────┤
│传输方式  │ 字节流   │ 数据报   │
├──────────┼──────────┼──────────┤
│一对一/多 │ 一对一   │ 一对一   │
│          │          │ 一对多   │
│          │          │ 多对多   │
├──────────┼──────────┼──────────┤
│应用场景  │ 文件传输 │ 视频直播 │
│          │ 邮件     │ 语音通话 │
│          │ HTTP     │ DNS查询  │
└──────────┴──────────┴──────────┘

TCP三次握手：
1. SYN：客户端→服务器（请求连接）
2. SYN+ACK：服务器→客户端（同意连接）
3. ACK：客户端→服务器（确认连接）

TCP四次挥手：
1. FIN：客户端→服务器（请求断开）
2. ACK：服务器→客户端（确认请求）
3. FIN：服务器→客户端（请求断开）
4. ACK：客户端→服务器（确认断开）

记忆口诀：
TCP可靠慢开销大，面向连接字节流
UDP不可靠快开销小，无连接数据报
三次握手建连接，四次挥手断连接
```

## 5.Z 典型例题精讲 ⭐⭐⭐⭐⭐

### 例题1：OSI七层模型 ⭐⭐⭐⭐⭐
路由器工作在OSI模型的哪一层？
A. 物理层  B. 数据链路层  C. 网络层  D. 传输层

**解析**：
```
网络设备工作层次：
- 物理层：中继器、集线器
- 数据链路层：网桥、交换机
- 网络层：路由器 ✓
- 应用层：网关

路由器功能：
- 路由选择
- IP地址识别
- 连接不同网络
- 工作在网络层

记忆：
- 物理层：信号放大
- 数据链路层：MAC地址
- 网络层：IP地址（路由器）
- 应用层：协议转换
```
**答案**：C

### 例题2：IP地址分类 ⭐⭐⭐⭐⭐
IP地址172.16.0.1属于哪一类？
A. A类  B. B类  C. C类  D. D类

**解析**：
```
IP地址分类：
- A类：1-126（第一个字节）
- B类：128-191（第一个字节）✓
- C类：192-223（第一个字节）
- D类：224-239（多播）
- E类：240-255（保留）

172在128-191之间，属于B类

B类地址特点：
- 网络位：16位
- 主机位：16位
- 默认子网掩码：255.255.0.0

记忆：
- 看第一个字节
- 1-126 → A类
- 128-191 → B类
- 192-223 → C类
```
**答案**：B
