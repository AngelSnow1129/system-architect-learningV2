## 🚀 使用方法

### 最简单的方式

**Windows:**
```bash
双击 start_server.bat
```

**Linux/Mac:**
```bash
./start_server.sh
```

**或直接打开:**
```bash
双击 web/index.html
```

### 访问地址
- **主页**：http://localhost:8000/web/
- **诊断**：http://localhost:8000/web/diagnostic.html
- **测试**：http://localhost:8000/web/test-loading.html
